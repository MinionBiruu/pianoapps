Privacy Policy – SimplePiano

Terakhir diperbarui: 09-03-2026

SimplePiano (“Aplikasi/Game”) dikembangkan oleh Minion Blue Studio (“Kami”). Kebijakan Privasi ini menjelaskan bagaimana Aplikasi memperlakukan data pengguna.

1. Data yang dikumpulkan

Aplikasi ini tidak mengumpulkan, menyimpan, atau membagikan data pribadi pengguna secara langsung.

Namun, Aplikasi menampilkan konten dari situs web pihak ketiga melalui WebView (misalnya: https://virtualpiano.online/). Situs tersebut dapat menggunakan cookie/teknologi serupa dan dapat mengumpulkan data sesuai kebijakan mereka. Silakan lihat kebijakan privasi situs terkait.

2. Izin (Permissions)

  • Internet: untuk memuat halaman web dan mengambil konfigurasi (remote config).

3. Remote Config

Aplikasi dapat mengambil konfigurasi (misalnya URL halaman yang dibuka) dari alamat yang kami kelola (contoh: GitHub Pages). Konfigurasi ini hanya digunakan untuk menentukan halaman web yang ditampilkan.

4. Cookie dan penyimpanan

WebView dapat menggunakan cookie/penyimpanan lokal untuk mendukung fungsi situs web yang dibuka (misalnya sesi atau preferensi). Cookie ini dikelola oleh situs web yang ditampilkan.

5. Keamanan

Kami berusaha menggunakan koneksi HTTPS saat memuat konten. Namun keamanan data juga bergantung pada situs web yang ditampilkan.

6. Perubahan kebijakan

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan dipublikasikan pada halaman ini.
